Paperloom
v0.2.1 · Web installer
Browser-flashable firmware

A reader that lives quietly on your shelf.

Paperloom is a paper-first reader for the LilyGo T5 E-Paper S3 Pro. Plug yours in below — your browser will do the rest. No drivers, no terminal, no PlatformIO.

Install the latest firmware

Connect your reader over USB-C and click Install. Takes about a minute, end to end.

This browser doesn't support Web Serial. Open this page in Chrome, Edge, or Opera on Mac, Windows, or Linux to install over USB. Web Serial needs a secure origin. Reload this page over https:// and try again. Use Chrome, Edge, or Opera on a desktop computer.

How it works

Four small steps. After this, your reader updates itself over WiFi — you'll never need this page again.

  1. i

    Plug in the e-reader

    Connect a USB-C cable that supports data (some thin cables only carry power and won't show up on your computer).

  2. ii

    Open the install dialog

    Click Install Paperloom above, then pick the serial port labeled USB JTAG/serial debug unit.

  3. iii

    Let it flash

    The installer writes the firmware in about 30–60 seconds. The device's screen will stay blank during this — that's normal.

  4. iv

    Press RST once

    The reader boots into Paperloom. From here, future updates arrive over WiFi via Settings → Firmware Update.

If something goes sideways

A short list of things to try before you give up. Most installer issues come down to the cable.

The device doesn't show up when I click Connect

Hold the BOOT button on the device, press RST once, then release BOOT. This forces the chip into download mode. Click Install Paperloom again and the port should appear.

"Failed to initialize" or the install stalls

The cable is the most common culprit. Try a different USB-C cable — ideally one that's known to carry data (the cable that came with a phone or laptop is a safer bet than a random charging cable).

I'm on Linux and the port isn't visible

Add your user to the dialout group, then log out and back in:

sudo usermod -aG dialout $USER

I'd rather flash it myself with esptool or PlatformIO

The firmware binaries (firmware.bin, the merged install image, and a friendly download copy) are attached to every GitHub release. Use whichever tool you prefer.

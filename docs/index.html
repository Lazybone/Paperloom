Paperloom

Browser-based firmware installer for your LilyGo e-reader.

Install firmware

Connect your LilyGo T5 E-Paper S3 Pro (or Pro Lite) to this computer over USB-C, then click below.

Your browser doesn't support Web Serial. Use Chrome, Edge, or Opera on desktop. Web Serial is disabled. Open this page over HTTPS in a supported browser.

Requires Chrome, Edge, or Opera on desktop. Firefox and Safari are not supported.

How it works

  1. Plug the e-reader into your computer with a USB-C cable.
  2. Click Connect and select the device port (look for USB JTAG/serial debug unit).
  3. Click Install Paperloom and wait for the flash to finish (~30–60 seconds).
  4. The device reboots into Paperloom automatically.

You only need this once. From then on, the device updates itself over WiFi via Settings → Firmware Update.

Troubleshooting

  • No device shows up? Press the small RST button on the device while it's plugged in, then click Connect again.
  • "Failed to initialize" error? Try a different USB-C cable — some cables are power-only.
  • Linux: add yourself to the dialout group (sudo usermod -aG dialout $USER) and re-login.
  • Manual flash: the firmware binaries are attached to every GitHub release if you prefer esptool.py or PlatformIO.
